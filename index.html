Affichez votre identité tech avec élégance.

Un widget léger, isolé via Shadow DOM, pour intégrer votre profil, votre stack et vos objectifs sur n'importe quel site web en une ligne de code.

⭐ Star sur GitHub
Capture d'écran du rendu final

Installation Rapide

Ajoutez simplement le script et initialisez-le avec vos informations.

<script src="/js/techcard.js"></script>
<script>
  new window.TechCard({
    card: "solo",
    user: {
      name: "Votre Nom",
      role: "Développeur Full-Stack",
      avatar: "url_image.jpg"
    }
  }).open();
</script>

Personnalisable

  • ✓ Thèmes : Support natif des modes Light et Dark.
  • ✓ Formats : Version "Mini" pour plus de discrétion ou "Solo" pour un profil complet.
  • ✓ Stack & Stats : Affichez vos technos préférées et vos chiffres clés.
