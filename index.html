Installation Rapide

Intégrez TechCard en quelques secondes. Copiez le snippet ci-dessous et collez-le avant la balise </body> de votre site. 

<!-- Import TechCard via CDN -->
<script src="https://cdn.jsdelivr.net/gh/Tostenn/TechCard/js/techcard.js"></script>

<script>
  window.onload = () => {
    new window.TechCard({
      card: "solo",
      user: {
        name: "Votre Nom",
        role: "Développeur Full-Stack",
        avatar: "https://votre-image.jpg"
      }
    });
  }
</script>

Personnalisable

  • ✓ Thèmes : Support natif des modes Light et Dark.
  • ✓ Formats : Version "Mini" pour plus de discrétion ou "Solo" pour un profil complet.
  • ✓ Stack & Stats : Affichez vos technos préférées et vos chiffres clés.
